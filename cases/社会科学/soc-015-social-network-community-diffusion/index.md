# 案例概述

本案例演示社会网络分析中的核心方法：社区发现、中心性分析和扩散模拟。通过构建模拟社交网络，探索社区结构如何影响信息扩散。

## 分析设计

- 网络：随机块模型（SBM）生成，含3个社区和组间/组内连接概率差异。
- 分析：度中心性、介数中心性、PageRank；Louvain社区发现。
- 扩散：简化SI模型，从高度中心节点出发，记录扩散范围随时间变化。

