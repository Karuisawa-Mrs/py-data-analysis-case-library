# 案例概述

本案例演示自然语言处理中的情感分析任务，对比传统机器学习方法（TF-IDF + 逻辑回归）与深度学习方法（BERT/Transformer）在公众反馈文本分类上的表现。

## 分析设计

- 数据：模拟公众反馈文本（正/负面），通过模板和关键词替换生成。
- 模型：TF-IDF + Logistic Regression（基线） vs BERT/distilBERT（高级，若依赖可用）。
- 评估：准确率、F1、混淆矩阵、 top 特征词或注意力权重。

## 复现说明

```bash
python analysis.py --smoke-test
python analysis.py
```

## 结果说明

输出包括模型指标表、分类报告、混淆矩阵图和可解释特征表。若Transformer库缺失，脚本自动降级至TF-IDF基线并注明。
