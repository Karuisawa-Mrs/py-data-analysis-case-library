# 案例概述

结构方程模型（SEM）适合处理“概念不可直接观测、但可由多个问卷题项反映”的社会科学问题。本案例把**社会资本**、**社会信任**和**主观幸福感**建模为 3 个潜变量，并用模拟问卷指标演示两步工作流：

1. 先用确认性因子分析（CFA）检查测量模型是否能把题项稳定映射到潜变量；
2. 再在同一 SEM 中估计“社会资本 → 社会信任 → 主观幸福感”的中介路径。

由于案例使用模拟数据，结果只用于说明潜变量建模、路径解释和中介分解的逻辑，不应被当作真实居民调查的经验结论。

## 分析设计

- 潜变量：
  - `social_capital`：由 3 个社会资本题项 `sc1`、`sc2`、`sc3` 反映；
  - `social_trust`：由 3 个社会信任题项 `st1`、`st2`、`st3` 反映；
  - `swb`：由 4 个主观幸福感题项 `swb1`、`swb2`、`swb3`、`swb4` 反映。
- 结构路径：
  - `social_trust ~ social_capital`
  - `swb ~ social_capital + social_trust`
- 中介解释：
  - 间接效应 = `social_capital → social_trust` × `social_trust → swb`
  - 总效应 = 直接效应 + 间接效应

## 结果说明

- `outputs/cfa_fit_stats.csv`：CFA 的拟合指标，如 CFI、TLI、RMSEA。
- `outputs/sem_fit_stats.csv`：完整结构模型的拟合指标。
- `outputs/parameter_estimates.csv`：测量模型和结构模型的参数估计表（含标准化系数）。
- `outputs/mediation_effects.csv`：直接效应、间接效应、总效应及 bootstrap 置信区间。
- `outputs/summary.txt`：文字摘要，解释样本、路径结果、中介结果与边界声明。
- `outputs/smoke_test.txt`：自动验证脚本使用的最小产物。
