# 案例概述

本案例演示如何将自然语言处理技术应用于社会科学研究中的政策文本分析。使用BERTopic（或回退至LDA）对模拟政策文档进行主题建模，识别核心议题并可视化主题分布。

## 分析设计

- 数据：模拟政策文档，每篇文档由预定义主题模板和随机词采样生成。
- 方法：BERTopic（基于Transformer嵌入 + HDBSCAN + c-TF-IDF）或回退至LDA（sklearn实现）。
- 输出：主题信息表、文档-主题映射、关键词表、主题 prevalence 图。

## 复现说明

```bash
python analysis.py --smoke-test
python analysis.py
```

## 结果说明

输出包括主题汇总表、文档映射表、关键词表和 prevalence 图。若BERTopic依赖缺失，脚本自动降级至LDA并注明。
