# 案例概述

本案例演示多层次（分层）线性模型在教育研究中的应用。通过模拟学生嵌套于学校的数据，探索个体层面与学校层面因素对学业成绩的影响。

## 分析设计

- 数据：模拟多层数据，含学生成绩、个体SES、学校资源、学校ID。
- 模型：空模型（估计ICC）→ 随机截距模型 → 随机斜率模型。
- 输出：固定效应表、随机效应表、方差分解表、预测值vs观测值图。

## 复现说明

```bash
python analysis.py --smoke-test
python analysis.py
```

## 结果说明

输出包括固定效应估计、随机效应估计、方差分解和拟合图。结果基于模拟数据，仅供方法教学。
