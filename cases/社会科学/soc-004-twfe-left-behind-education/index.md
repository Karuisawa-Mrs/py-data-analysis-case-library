# 案例概述

该案例用**模拟的学生-年份面板数据**演示：当研究问题关心“父母外出务工导致的留守状态变化”与“同一名学生成绩变化”之间的关系时，双向固定效应（Two-Way Fixed Effects, TWFE）如何同时控制学生固定效应与年份固定效应。识别重点不是横向比较“谁更容易成为留守儿童”，而是利用**同一学生在不同年份中的状态切换**来识别成绩波动。

由于数据是模拟生成的，案例重点放在方法逻辑、估计流程与结果解释边界，而不是给出现实世界的经验结论。`analysis.py` 会生成主回归、聚类稳健标准误，以及按年龄组与地区组拆分的异质性估计结果。

## 分析设计

- 观测单位：`student_id × year` 面板。
- 被解释变量：标准化后的期末考试总分 `exam_score`。
- 核心解释变量：`left_behind`，表示该年父母外出务工导致的留守状态。
- 固定效应：
  - **个体效应**：吸收学生不随时间变化的能力、家庭文化资本、先天气质等不可观测因素。
  - **年份效应**：吸收所有学生共同面对的政策、考试难度、宏观冲击与教育环境变化。
- 协变量：家庭收入、学习时长、监护支持、学校资源等时变因素。
- 异质性：分别在低龄/高龄学生与中西部/沿海地区子样本上重复估计。

## 复现说明

```bash
python analysis.py --smoke-test

```
