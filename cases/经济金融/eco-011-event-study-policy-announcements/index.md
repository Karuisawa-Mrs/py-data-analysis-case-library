# 案例概述

本案例模拟政策公告对股票收益的影响，通过事件研究法（Event Study）计算异常收益（Abnormal Returns, AR）和累积异常收益（Cumulative Abnormal Returns, CAR），并进行统计显著性检验。

## 分析设计

- 数据：模拟个股日收益与市场指数收益，随机设定政策公告日。
- 方法：市场模型（Market Model）估计正常收益，计算AR与CAR。
- 检验：t检验判断CAR在事件窗口内是否显著异于零。
