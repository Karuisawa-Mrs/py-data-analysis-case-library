# 案例概述

本案例演示因果推断与机器学习的交叉方法：双重机器学习（Double ML）和因果森林（Causal Forest）。在高维协变量环境下，估计政策的平均处理效应（ATE）并探索异质性处理效应（CATE）。

## 分析设计

- 数据：模拟面板数据，包含处理变量、 outcome、高维协变量和混淆因素。
- 方法：Double ML（交叉拟合 + Neyman正交得分）和Causal Forest（基于随机森林的CATE估计）。
- 输出：ATE/ATT点估计与置信区间、按子群体划分的CATE表、正交得分诊断。

## 复现说明

```bash
python analysis.py --smoke-test
python analysis.py
```

## 结果说明

输出包括ATE/ATT估计表、CATE子群体表、正交得分诊断文件和CATE分布图。若econml未安装，脚本使用sklearn回退实现简化版交叉拟合DML。
