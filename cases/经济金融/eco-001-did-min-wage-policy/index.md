# 案例概述

本案例用企业面板模拟数据演示最低工资政策的经典双重差分（Difference-in-Differences, DID）识别逻辑。场景设定为：一部分企业所在地区提高最低工资，另一部分相似企业所在地区未调整政策；我们比较两组企业在政策实施前后的就业变化差异，以估计政策处理效应。

案例目标不是给出现实世界最低工资政策的结论，而是帮助读者掌握 DID 的核心构件：处理组/对照组划分、政策前后期定义、平行趋势可视化、固定效应回归估计与安慰剂检验。

## 分析设计

- 样本为企业-时期面板数据，包含处理组与对照组、政策前后期、企业固定特征及若干观测控制变量。
- `analysis.py` 先按固定 seed 生成模拟数据，再输出平行趋势图与 DID 回归结果。
- 主回归采用企业固定效应与时期固定效应框架，政策效应由 `treated × post` 交互项识别。
- 安慰剂检验将“伪政策时点”放在真实政策前期，验证不存在虚假的提前处理效应。

## 复现说明

- `outputs/parallel_trends.png`：处理组与对照组的平均就业趋势图，用于检查政策前走势是否近似平行。

- `outputs/placebo_results.csv`：安慰剂检验结果。
- `outputs/summary.txt`：对估计值、解释边界与复现结果的文字摘要。
