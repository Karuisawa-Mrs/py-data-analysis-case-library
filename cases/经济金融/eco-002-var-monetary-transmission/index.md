# 案例概述

本案例围绕“货币政策如何传导到实体经济与价格变量”这一经典宏观问题，使用一个轻量、可复现的模拟宏观系统来演示向量自回归（VAR）分析流程。脚本生成五个时间序列：`M2`、`GDP_deflator`、`investment`、`consumption`、`exchange_rate`，并将其组织为一个适合教学的多变量动态系统。

案例重点不是识别真实世界的央行外生冲击，而是演示：当我们把货币变量创新当作一个“示范性政策冲击”时，如何通过平稳化、滞后阶数选择、脉冲响应函数（IRF）与预测误差方差分解（FEVD）来描述动态传播路径。

## 分析设计

- 原始模拟序列先以水平值生成，再通过对数差分转为近似增长率或价格增速，避免直接对趋势水平变量估计 VAR。
- `analysis.py` 会输出 `stationarity_checks.csv`，同时比较水平值与对数差分值的 ADF 检验结果，用于说明“为什么要先做平稳化处理”。
- 滞后阶数通过 `statsmodels.tsa.api.VAR.select_order` 基于信息准则自动选择，默认记录 AIC/BIC/HQIC/FPE 的结果并输出到 `lag_order_selection.csv`。
- IRF 使用变量排序 `M2 → GDP_deflator → investment → consumption → exchange_rate` 的 Cholesky 正交化解释，因此结果依赖变量排序，只能作教学示范解读。
- FEVD 用于比较在给定预测期内，各变量的预测误差中有多大比例可由 `M2` 冲击解释。

## 复现说明

```bash
python analysis.py --smoke-test
python analysis.py
```

Notebook 只负责调用脚本并展示 `outputs/` 下的文字结果与图形结果，不承载独立分析逻辑。

## 结果说明

- `outputs/summary.txt`：记录平稳化说明、滞后阶数选择、IRF 峰值响应与 FEVD 终点贡献度。
- `outputs/irf_m2_shock.png`：展示对 `M2` 正交化冲击的响应路径。
- `outputs/fevd_overview.png`：展示各变量预测误差方差的来源分解。

在解释上，建议把它理解为“示范性货币冲击传播脚手架”：它适合教学和方法演示，但不适合直接得出真实央行政策效应大小的经验结论。
